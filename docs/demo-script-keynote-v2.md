# 第三幕 · 它是怎么做到的

## 2:15 — 2:55 · Hermes

◐ Galaxy 全景出现。深空背景，紫色光簇缓缓自转。镜头不动。

◇ 屏幕右上角浮出一张极简卡片，无装饰边框，仅三行字：

```
在 AXIOM 内部，
有一个我们自己写的系统。

它叫 Hermes。
```

‖ 一秒 ‖

▮：

> 在 AXIOM 的内部，
> 有一个我们自己写的系统。
>
> 它叫 **Hermes**。

◇ 卡片向下扩展，又增三行：

```
不基于 LangChain。
不基于 AutoGen。
不基于任何开源框架。
```

▮：

> 不基于 LangChain。
> 不基于 AutoGen。
> 不基于任何现成框架。

‖ 一秒 ‖

◇ 卡片缓慢翻面，呈现三层结构。每一层依次点亮，自上而下：

```
        Hermes



  ⓐ  Agent A · 教
  ⓑ  Agent B · 听



  Oracle    Profile    Forge
  Guide     Assess



  Orchestration · Memory · Cognition
```

▮（节奏明快）：

> 它分三层。
>
> 最上面，是两个并行的 Agent——
>
> 一个，专心**教**你。
> 另一个，在背后，专心**听**你。
>
> 中间，是五个专业角色。
> 像一个教研组，随时待命。
>
> Oracle，是导师。
> Profile，是画像分析师。
> Forge，是卡片锻造师。
> Guide，是学习向导。
> Assess，是评估专家。
>
> 最下面，是编排、记忆，与认知。
>
> 我们没有用任何现成框架。
>
> 因为没有任何现成框架，
> 能让一个 AI 在教你的同时，
> 让另一个 AI ，
> 默默地，记住你。

# 附录 D · 拍摄关键提示

1. **开场禁止音乐**。前 45 秒只有键盘环境音、远处低频底噪。音乐从 AXIOM 字揭幕处开始——**深邃，克制，低频，无旋律性**。
2. **字幕字号必须大，字数必须少**。每屏最多 3 行，每行不超过 14 字。
3. **黑屏不是过渡，是一个镜头**。镜头切黑超过 1 秒时，演讲者必须停下。
4. **结尾字独占一屏，4 秒内不出现任何其他元素**。最后才浮现赛题信息。
5. **整片唯一允许的「我们」**：在戒律段。其他地方一律说"它"和"你"。**把产品当人讲，把观众放进去。**
6. **Hermes 架构翻面那一下，必须是预录的视频片段**，不要现场触发。
7. **不要在任何一处出现"接下来"、"我们再看一下"、"这里我们可以看到"** 这类导览口播。
8. **拍完做一件事**：把所有旁白删掉，只看画面和字幕。如果画面讲不通——不是画面的问题，是脚本不够好。
