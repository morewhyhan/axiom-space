## 视频总览

**总时长：6 分 50 秒**
**风格：一镜到底，单镜头连续操作，用旁白+字幕交代无法UI直接展示的部分**
**核心叙事线：一个学生从进入系统 → 对话 → 背后Agent自动工作 → 知识体系逐步构建的完整旅程**

**画面**：黑屏。白字逐行浮现，背景是散落的碎片粒子缓慢飞散。

```
第一行（第0-12秒）：
我们学了很多，但都是碎片。
没有输出，没有关联，没有积累。

第二行（第13-22秒）：
教育心理学 Bloom 2 Sigma 告诉我们：
一对一导师 + 掌握学习法 = 超过 98% 的学生。

第三行（第23-30秒）：
但一对一导师太贵了。
AI 是唯一的解法。
可 AI 产品一打开就是一个空白输入框——
我们最大的问题是：不知道自己不知道什么。
```

**旁白**（沉稳、慢速、每句之间留 0.5 秒呼吸）：
> 大学生学了很多……但都是碎片。
> 教育心理学家 Bloom 四十年前就发现了：
> 一对一导师，加上掌握学习法，就能让你的成绩超过 98% 的人。
> 可一对一导师太贵了。
> AI 是唯一的希望 —— 但打开任何一个 AI 产品，就是一个空白输入框。
> 而人类最大的困境恰恰是：**我不知道我不知道什么。**

## §4 【2:30 — 3:45】多智能体协同 + 5种子Agent + 6种资源（75 秒）

**操作**：回到 Forge，用户继续输入：
> "帮我系统理解第一性原理 —— 概念、历史案例、和我编程工作有什么关系？"

**关键镜头 — Orchestration Engine 可视化**：
屏幕右侧展开 **编排引擎面板**（演示用，反映真实后端 `orchestration-engine.ts` 的流程）：

```
┌─ Hermes Orchestration Engine ───────────┐
│  工作流：resource_generation              │
│  消息总线：MessageBus (EventEmitter)      │
│                                          │

│  ⚪ [Planner Agent] 拆解任务清单          │
│     ↓ 发送 task 到 Generator (并行)      │
│  ◉ [Generator Agent] 6 路并行生成中      │
│     ├─ 讲解文档 ████████░░ 80%           │
│     ├─ 思维导图 ██████░░░░ 60%           │
│     ├─ 习题集   ████████░░ 80%           │
│     ├─ 代码实操 █████░░░░░ 50%           │

│     ┌──────────────────────────────┐    │

│     └──────────────────────────────┘    │

│                                          │
│  总耗时：12.4s · Agent 协作轮次：5       │
└──────────────────────────────────────────┘

**旁白**（这一段用于体现技术深度，语速稍快但不急）：
> 这背后是 Hermes 的 **第二层：5 种子 Agent 角色在协同工作**。
> Profile 读你的画像，Planner 把"系统理解第一性原理"拆成可执行任务，
> Generator **6 路并行生成**，然后 Reviewer **做质量把关**，
> 最后 Pusher 一并推送到你的文献盒。
> 全部通过**消息总线**通信 —— 这是我们自己写在 `orchestration-engine.ts` 里的编排系统。
> 市面上大多数所谓"多智能体"只是一个 LLM 调不同的 prompt ——
> AXIOM 是**有框架、有角色分工、有质检流程的完整系统**。

**操作**：生成完成 → 切到 Galaxy → 6 颗粉色文献节点依次飞入 → 逐个悬停展示资源：

| # | 类型 | 真实内容预览 |

| 4 | **代码实操** | Python 第一性原理思维应用示例 |
| 5 | **拓展阅读** | 3 篇精选论文+书籍章节 |
| 6 | **图解 SVG** | 两种思维模式对比图 |

**旁白**：
> **6 类资源，全部基于你的画像定制。**
> 每一条都有**引用出处、RAG 检索验证、事实核查通过**。
> 这就是我们的防幻觉体系 —— 不是随口说，是**可追溯、可验证**。

**字幕**：
```
· ✓ 5 种子 Agent：Oracle · Profile · Guide · Forge · Assess
· ✓ 自研 Orchestration Engine + EventEmitter 消息总线

5. **Cognitive Layer** 包括记忆管理、上下文压缩、模式检测，不是简单的 vector store
