# 基本灵感构建

**项目名称：** 领域巨匠对话式AI个性化知识构建系统

***

## 一、项目核心定位

本项目为**Web端AI驱动的个性化知识学习系统**，以**卢曼卡片盒笔记法**为底层逻辑，通过与各领域学术巨匠（如冯·诺依曼等诺贝尔奖得主、学科奠基人）的Agent对话，完成用户知识边界探测与动态画像构建；依托**三盒体系**完成知识从输入、内化到系统化输出的全流程，AI全程辅助提质提速，最终打造用户专属的精准概念知识网络，实现轻量化、沉浸式、因材施教的深度学习。

***

## 二、核心设计理念

1. **摒弃传统表单式画像采集**，以学科巨匠Agent对话为核心入口，通过自然聊天记录深度捕捉用户知识基础、认知边界、学习偏好。
2. **严格复刻卢曼三盒体系**，不增设冗余功能，所有学习行为闭环于闪念盒、文献盒、永久盒之中。
3. **采用数据库存储**，所有卡片、对话记录、用户画像统一管理，支持高效检索与关联分析。
4. **AI仅做辅助引导与资料供给**，核心概念卡片由用户自主构建，保证知识内化质量，同时实现学习效率的无损提速。

***

## 三、系统完整运行闭环

### 1. 巨匠Agent创建与对话初识

选定学习领域后，系统自动生成对应领域的学科奠基人/学术巨匠Agent（如学习计算机科学则创建冯·诺依曼Agent）；用户与该Agent自然对话聊天，聊天全程作为AI探测用户知识边界、学习目标、认知水平的核心依据，完成初始动态画像构建。

### 2. 个性化文献资料生成

AI基于用户与巨匠Agent的对话记录，分析用户知识盲区与学习需求，生成专属多模态学习资料清单（书单、视频、文档、图解等），自动存入**文献盒**，作为用户学习的核心参考素材。

### 3. 学习内化与闪念卡片记录

用户研读文献盒内资料，结合与巨匠Agent的交流感悟，将零散想法、瞬时思考、知识疑问录入**闪念盒**，无需格式约束，仅做碎片化知识留存。

### 4. 概念打磨与永久卡片沉淀

AI引导用户对闪念盒内碎片内容进行梳理、完善，朝着「清晰、准确、必要」的标准打磨；最终形成的标准化概念卡片，自动建立双向关联并存入**永久盒**，成为用户对该领域知识的系统化反馈与内化输出。

### 5. AI深度迭代与学习提速

AI读取永久盒内用户自主构建的概念卡片，深度理解用户知识体系与成长轨迹，动态更新学生画像；基于最新画像优化巨匠Agent对话逻辑、补充更精准的文献资料，引导用户搭建更多关联概念，实现不降低质量的学习加速，形成持续成长的学习闭环。

***

## 四、卢曼三盒体系落地

### 1. 闪念盒（Fleeting Notes）

存储用户学习中的零散思考、感悟与疑问，无格式要求，是知识输入的原始载体。

### 2. 文献盒（Literature Notes）

存储AI基于对话生成的多模态学习资料，包含各类参考素材、视频链接、文档解析，用于填补用户知识盲区。

### 3. 永久盒（Permanent Notes）

存储经打磨后的标准化概念卡片，卡片间建立双向关联，形成用户专属的系统化知识网络，以知识图谱作为可视化索引。

***

## 五、多智能体架构设计

### 1. 领域巨匠Agent

核心交互载体，对应不同学科领域，通过对话完成用户知识边界探测与初始画像构建。

### 2. 文献生成Agent

基于用户画像与知识盲区，生成个性化多模态学习资料，对接文献盒。

### 3. 卡片构建Agent

引导用户打磨闪念内容，生成标准概念卡片并建立关联，对接永久盒。

### 4. 迭代优化Agent

分析永久盒知识体系，更新用户画像，提速学习流程，实现全系统动态优化。

***

## 六、存储与技术架构

1. **Web应用架构**，前端采用next.js，后端采用Hone.js，数据库采用sqlite。
2. **数据库存储设计**，三盒体系、对话记录、用户画像均以结构化数据存储，支持高效查询与关联分析。
3. **知识图谱可视化**，基于卡片间的双向关联关系，动态渲染用户专属知识网络，支持交互式探索。

***

## 七、赛题适配核心亮点

三盒体系运行闭环
```
  对话初识 → 文献盒(资料生成) → 闪念盒(零散思考) → 永久盒(概念卡片) → 迭代优化
     ↑                                                            ↓
     └──────────────── 持续闭环，越学越智能 ──────────────────────┘
```

| 赛题要求        | 方案适配                 |
| ----------- | -------------------- |
| 对话式6+维度动态画像 | 通过巨匠Agent对话自主构建，实时更新 |
| 多智能体协同架构    | 多Agent分工配合生成多模态学习资料  |
| 个性化学习路径规划   | 基于三盒体系自然融入学习流程       |
| 智能辅导        | 巨匠Agent持续对话答疑        |
| 学习效果评估      | 永久盒知识体系分析，动态优化       |

1. 满足对话式6+维度动态画像要求，通过巨匠Agent对话自主构建，实时更新。
2. 实现多智能体协同架构，多Agent分工配合生成多模态学习资料。
3. 天然融入个性化学习路径、智能辅导、学习效果评估，所有功能均内嵌于三盒学习流程，无刻意设计。
4. 系统具备持续成长性，知识卡片越完善，AI适配越精准，学习效率与知识体系化程度同步提升。
