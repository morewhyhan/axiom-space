## 连续视频舞台

第 04—16 页共用页面外的同一个 `#continuityStage`。第 04 页先以 `seed` 形态从系统架构中出现；第 05 页再展开为实录。翻页时只改变位置、尺寸、裁切比例、场景标签和时间点，因此视觉上是一个视频对象持续变形，而不是每页重新出现一个播放器。

舞台预设：

- `right`：右侧主画面，左侧解释
- `left`：左侧主画面，右侧解释
- `wide`：中央宽银幕，用于关键动作
- `full`：近全屏，作为章节性视频转场
- `focus`：右侧聚焦，用于从图谱点入灵感卡
- `float`：右下浮窗，用于机制回扣
- `ribbon`：顶部横幅，用于价值页证据蒙太奇

当前没有母视频时，舞台显示随场景变化的产品 UI 占位动画。未来只需：

1. 把一镜到底母视频命名为 `axiom-one-take.mp4`，放入 `media/`；
2. 页面会自动读取 `./media/axiom-one-take.mp4`，不需要再改视频组件；
3. 若后期剪辑改变时间轴，只需调整各页的 `data-video-time` 秒数。

同一个 `<video>` 元素会跨页保留，不需要为每页复制视频。

## 操作

- `← / →`、滚轮、触屏：翻页
- `ESC`：索引视图
- `B`：低功耗 / 静态模式
- `Home / End`：首尾页

## 逐页配音

- 17 页均绑定独立 MP3，文件位于 `assets/audio/`，同时保留对应 VTT 字幕。
- 使用微软神经中文女声 `zh-CN-XiaoxiaoNeural` 生成，参数为语速 `-4%`、音高 `-2Hz`、音量 `+0%`。
- 总口播时长约 `429.7s`，即 `7:09.7`；音频统一为 `48kHz`、单声道、`128kbps`，并完成响度归一化。
- 页面不提供额外的配音播放按钮：打开时先自动尝试播放；按方向键、点击页码或触屏翻到新页时，会停止上一页并立即播放当前页配音。
- 浏览器若拦截刚打开页面时的有声自动播放，第一次正常翻页动作会直接完成解锁，不需要再点一次“播放”。
- 产品实录视频始终静音自动播放，避免与配音抢声道。
- 最终逐页文字稿见 `docs/03-宣传资料/AXIOM-Space-A3黄金案例-V21-逐页最终配音稿.md`。
